| **Section**                                | **Details**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Project Name**                           | Cab Booking System – Java 8 + Hibernate 5 + MySQL + Maven                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Description**                            | Console-based CRUD application for Users, Cabs & Bookings using DAO layer and Hibernate ORM.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Technologies Used**                      | Java 8, Hibernate 5.6, Maven, MySQL, javax.persistence API                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Folder Structure**                       | `src/main/java/com/cabbooking` (Main App), `models/` (Entities), `dao/` (Hibernate + DAO), `src/main/resources/hibernate.cfg.xml`, `database/database.sql`                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Software Requirements**                  | JDK 8, Maven, MySQL 8/5.7, Eclipse/IntelliJ (with Maven support)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Maven Dependencies**                     | Hibernate Core (5.6.15.Final), javax.persistence-api (2.2), MySQL Connector-J (8.0.33), slf4j-simple (2.0.9)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **pom.xml Essentials**                     | `xml<br><dependency><groupId>org.hibernate</groupId><artifactId>hibernate-core</artifactId><version>5.6.15.Final</version></dependency><br><dependency><groupId>javax.persistence</groupId><artifactId>javax.persistence-api</artifactId><version>2.2</version></dependency><br><dependency><groupId>mysql</groupId><artifactId>mysql-connector-java</artifactId><version>8.0.33</version></dependency><br><dependency><groupId>org.slf4j</groupId><artifactId>slf4j-simple</artifactId><version>2.0.9</version></dependency>`                                                                                              |
| **Database Setup**                         | 1. Open MySQL<br>2. Run:<br>`sql<br>CREATE DATABASE cabdb; USE cabdb;<br><br>CREATE TABLE users ( id INT AUTO_INCREMENT PRIMARY KEY, name VARCHAR(100), email VARCHAR(100), phone VARCHAR(20) );<br><br>CREATE TABLE cabs ( id INT AUTO_INCREMENT PRIMARY KEY, number_plate VARCHAR(20), driver_name VARCHAR(100), model VARCHAR(100), available BOOLEAN DEFAULT TRUE );<br><br>CREATE TABLE bookings ( id INT AUTO_INCREMENT PRIMARY KEY, user_id INT, cab_id INT, pickup VARCHAR(255), dropoff VARCHAR(255), status VARCHAR(50), FOREIGN KEY (user_id) REFERENCES users(id), FOREIGN KEY (cab_id) REFERENCES cabs(id) );` |
| **Hibernate Config (`hibernate.cfg.xml`)** | `xml<br><property name="hibernate.connection.url">jdbc:mysql://localhost:3306/cabdb</property><br><property name="hibernate.connection.username">root</property><br><property name="hibernate.connection.password">root</property><br><property name="hibernate.dialect">org.hibernate.dialect.MySQL8Dialect</property><br><property name="show_sql">true</property><br><property name="hbm2ddl.auto">update</property><br><mapping class="com.cabbooking.models.User"/><br><mapping class="com.cabbooking.models.Cab"/><br><mapping class="com.cabbooking.models.Booking"/>`                                               |
| **Entity Classes**                         | **User.java, Cab.java, Booking.java** (all using `javax.persistence.*` with @Entity, @Id, @Table annotations).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **DAO Classes**                            | **UserDAO** (save, get, listAll, delete), **CabDAO** (save, get, listAll, update, delete), **BookingDAO** (save, get, listAll, delete)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **HibernateUtil.java**                     | `java<br>public static void shutdown(){ getSessionFactory().close(); }`                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Main Application (Main.java)**           | Console menu with options: Add User, List Users, Add Cab, List Cabs, Book Cab, View Bookings, Delete Booking, Exit. Uses Java 8 switch-case syntax.                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **How to Import in Eclipse**               | 1. Open Eclipse → **File → Import → Maven → Existing Maven Project**<br>2. Select project folder<br>3. Right-click project → **Maven → Update Project (Alt+F5)**<br>4. Right-click project → **Build Path → Configure Build Path** → Ensure JavaSE-1.8<br>5. Run Main.java                                                                                                                                                                                                                                                                                                                                                  |
| **How to Run**                             | Right-click `Main.java` → **Run As → Java Application** → Console menu appears with CRUD options                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Console Menu Options**                   | 1. Add User<br>2. List Users<br>3. Add Cab<br>4. List Cabs<br>5. Book Cab (marks cab unavailable)<br>6. View Bookings<br>7. Delete Booking (cab becomes available again)<br>8. Exit & Hibernate session shutdown                                                                                                                                                                                                                                                                                                                                                                                                            |
| **Troubleshooting**                        | **Unknown DB cabdb** → Create DB manually<br>**javax.persistence not found** → Run Maven Update<br>**Arrow case error** → Use Java 8 switch-case version<br>**shutdown() undefined** → Add method in HibernateUtil.java                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Credits**                                | Developed with Hibernate + MySQL + Maven, console-based UI.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
